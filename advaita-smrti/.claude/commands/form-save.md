# Save Form Answer

Write the finalized answer to `.sea_answer.md` for the Python assistant to capture.

## Instructions

1. Take the answer we've collaboratively developed
2. Write ONLY the answer text to `.sea_answer.md`:
   - No markdown headers
   - No "Final Answer:" prefix
   - No meta-commentary
   - Just the clean answer text, ready to paste into the application
3. Confirm to the user that the answer has been saved
4. Remind them to exit Claude Code (Ctrl+C or /exit) to return to the Python assistant

## Format

The file should contain exactly what should go in the application form - nothing more, nothing less.

## Example

If the question is "What is your 6-12 month goal?", the file might contain:

```
Within 6 months, I will secure 2-3 consulting clients generating $8,000/month in revenue. Key milestones:
- Month 1-2: Launch website and LinkedIn presence
- Month 3: First paying client through network referrals
- Month 6: Second recurring client, begin content pipeline

By month 12, I aim for $12,000/month with a mix of project work and retainer clients, plus 2 published technical articles establishing thought leadership.
```
